이전 1단계 결재 (2026-05-07) 8건이 운영 룰의 기본 방향을 설정했고, 이번 2단계 결재 (2026-05-08) 8건은 그 **세부 운영 방식**을 확정. CFO 자문 Sage가 단순 이분법 대신 **차등 적용** 패턴을 권장 → 평균 마진 7% 방어 + 영업 관계 보호 + 운영 속도 동시 확보.

## 🔴 우선 결재 — 매출·마진 직결

### 1. PG 수수료 흡수 모델 — 차등 적용

| 케이스 | 처리 | 효과 |
|--------|------|------|
| **Non-refundable** (호텔 마진 8~12%) | **100% 흡수 + 마진 1.5%p 추가 확보** | 노쇼 리스크 차단 |
| **Free Cancel** | 송금 유지 (PG 0) / 카드결제 시 정가 | UX 단순 |
| **표시가** | 단일 가격 (혼란 방지) | 내부 운영 차등 |

**효과**: 평균 마진 7% 방어 + 노쇼 동시 차단

### 2. 환차손 헷징 정책 — 자연헷지 우선

| 단계 | 수단 | 비용 |
|------|------|------|
| 1단계 | JPY 매출 → JPY 비용 매칭 (KR-JP 송금 최소화) | **0** |
| 2단계 | 잔여 USD/JPY 노출분 **3개월 롤링 Forward** | **0.3~0.5%** |
| 배제 | Option (비용 대비 효익 낮음) | — |

**운영**: 분기별 헷지 비율 리밸런싱, CFO 운영

## 🟡 신용·AR 운영 룰

### 3. 신용 한도 동결 트리거 — 2단계

| Aging | 조치 | 강도 |
|-------|------|------|
| **60일** | 신규 예약 신용 한도 **50% 축소** (소프트 동결) + 협의 요청 | 중 |
| **90일** | **완전 동결** + 법무 검토 트리거 | 강 |

**근거**: 단일 60일은 영업 관계 손상, 단일 90일은 회수 지연. 단계적 압박으로 균형.

### 4. Net-45 적용 결재 방식 — TTV 규모별 차등

| 월 TTV | 결재 |
|--------|------|
| **5억원+ 대형사** | 케이스별 **대표이사 결재** |
| **5억원 미만** | Master 위임 + 분기 보고 |

**원칙**: 60일 절대 금지 (재확인). Net-30 → Net-45 변경 사유 + 매출 임계 충족 증빙 필수.

## 🟢 디포짓 / 분쟁

### 5. No Deposit "최상위 신뢰 회사" 기준 — 정량 + 정성 하이브리드

**정량 3종 (모두 충족)**
| 지표 | 임계값 |
|------|--------|
| 월 TTV (12개월 평균) | **10억원+ (KRW)** |
| 거래 기간 | **24개월+** 연속 |
| 60일+ Overdue | **0건** (이력) |

**정성 2종 (대표이사 판단)**
- 회사 평판 (상장사·대형사·브랜드 가치)
- 대표이사 면담

**갱신**: 12개월마다 재평가

### 6. No Deposit 결재 양식 — 자동 평가서 통합

**자동 항목** (시스템 수집)
- 회사명 / 회사 ID
- TTV 12개월 (자동 계산)
- 거래기간 (자동)
- Aging 이력 (자동)

**수동 항목** (작성)
- Master 추천 사유
- 대표이사 의견 / 결재일

**워크플로우**: No Deposit 신청 → 자동평가 → 결재 → ELLIS Admin 등록

### 7. 분쟁 결재선 — 금액대 4단계

| 금액 (USD) | 결재선 | SLA |
|-----------|--------|-----|
| **< $1,000** | Master 단독 | **당일** |
| **$1,000 ~ 10,000** | Master + Accounting (양인) | **24h** |
| **$10,000 ~ 50,000** | Accounting + EllisOP | **48h** |
| **$50,000+** | **대표이사 결재** | **72h** |
| 시스템 분쟁 (Sanha 등) | 별도 트랙 | — |

### 8. 분쟁 SLA — 영업시간 기준 + Critical 별도

| 우선순위 | 응답 | 해결 | 비고 |
|---------|------|------|------|
| **High** | 4h | 24h | 영업시간 (KR/JP 9-18시) |
| **Medium** | 24h | 72h | 영업시간 |
| **Low** | 72h | 5영업일 | 영업시간 |
| **Critical** (부정결제·시스템장애) | 즉시 | 24x7 | **별도 트랙** |

**커버**: KR + JP 운영 = 9시간 → 한국 야간은 JP 인력 활용
**증원 트리거**: 월 분쟁 50건 초과
