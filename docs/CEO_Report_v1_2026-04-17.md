## 1. 프로젝트 개요

DOTBIZ는 OhMyHotel의 B2B 호텔 예약 포털로, 기존 DIDA B2B 포털을 벤치마킹하여 자사 플랫폼으로 구축하는 프로젝트입니다. 현재 프론트엔드 프로토타입 단계로, 전체 UI/UX를 구현하여 실제 서비스와 동일한 사용자 경험을 검증하고 있습니다.

### 3.3 PREPAY / POSTPAY 정산 구조 (신규 구현)

**닷비즈 핵심 비즈니스 구조를 프로토타입에 반영:**

| 구분 | POSTPAY (후불) | PREPAY (선불) |
|------|---------------|--------------|
| 디포짓 | Floating Deposit 등 6종 | 없음 |
| 정산 주기 | Monthly/Bi-weekly/Weekly | 즉시 결제 |
| 예약 생성 | Create → 후불 정산 | Pay & Book → PG 카드결제 |
| Non-refundable | 예약 가능 (후불) | 결제 후 예약 가능 |
| 카드 저장 | — | 첫 결제 시 저장 문의 → 재사용 |

**공통 정책 구현:**
- 예약 수정 불가 (Amend 버튼 제거)
- 취소 가능 예약 → 취소 후 재예약
- Non-refundable → 취소 불가
- 요청사항 변경 → 티켓 시스템으로만
