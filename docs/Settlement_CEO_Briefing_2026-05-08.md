## 2. 배경 — 왜 재검토가 필요한가

### 2-1. 마진 가정 변화

이전 결재(2026-05-07)는 **Gross Margin 7%** 가정 하에 수립됨.

대표이사 추가 의견 (2026-05-08): **실제 마진은 3.5~4%**.

| 항목 | 이전 가정 (7%) | 실측 (3.5~4%) |
|------|--------------|--------------|
| Gross Margin | 7.0% | **3.5~4.0%** |
| − ELS 적립 | -0.5~0.75% | 동일 |
| − CMS | -1.0% | 동일 |
| − PG 수수료 (글로벌 PG 흡수 시) | -3.5% | -3.5% |
| **Net Margin** | +1.75~2.25% | **-1.94~-1.44% 🚨 역마진** |

→ **글로벌 PG 흡수 시 역마진 발생 확정. 결제 전략 전면 재검토 불가피.**

### 2-2. 8가지 보완 의견 (CEO)

1. 우리 마진은 3.5~4%
2. CMS 1% + ELS 0.75% 시 마진 압박
3. 카드수수료까지 흡수 시 역마진
4. Eximbay 같은 글로벌 PG는 3.5~4%
5. 로컬 PG는 상대적으로 저렴
6. 국가별 PG 분리 검토 — 마켓 리서치 필요

## 4. 분석된 전략 옵션 (방향성)

### 옵션 A: 권역별 로컬 PG 다각화 (초기 검토)

각 권역에 Local PG 도입: 한국 Toss / 일본 GMO / 베트남 VNPay / 중국 Alipay 등

**장점**: 권역별 평균 수수료 3.5% → 2% 절감 (1.5%p ↑)
**단점**:
- 본사·지사 분리 회계 복잡도 증가 (계약/입금 주체는 항상 싱가포르)
- 각 PG 통합·협상·유지 비용 발생
- PCI-DSS 인증 필수 (자체 카드 정보 보유)
- 사업 확장 범위 과대

→ **❌ 채택 안 함**

### 옵션 B: 기존 Eximbay 유지 + 송금 유도 (CEO 방침)

선결제 PG는 Eximbay 단일 유지. 카드수수료는 판매가와 별도로 고객 부담.

기본 전략: 예약량 ↑ → 해외 송금(SWIFT) 활성화.

**장점**:
- 계약/입금 주체 = 싱가포르 단일 (회계 단순)
- Eximbay 다통화 정산으로 모든 권역 커버
- 신규 PG 협상·통합 비용 0
- 카드수수료 고객 부담 → DOTBIZ 마진 영향 0

**단점**: 권역별 PG 절감 효과 포기 (1.5%p)

→ **✅ 채택**

### 옵션 C: 중국 예외 — Alipay/WeChat 도입

옵션 B 기조 유지하되, **중국 시장만 예외**.

**사유**:
- 중국은 Alipay/WeChat이 결제 표준 (카드보다 보편적)
- Eximbay만으로는 중국 시장 진출 어려움
- 홍콩 지사(싱가포르 출자) 오픈 시 협상 주체 확보 가능

→ **✅ 채택** (홍콩 지사 오픈 결재 필요)

|------|------|

- 수금: 일반 거래는 SG 본사 직접 / 대형 거래는 SWIFT 송금

### 5-3. 신용·디포짓·분쟁 (총 16건 결재 완료)
